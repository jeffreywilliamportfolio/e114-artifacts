# Release Notes

## Initial Release

This bundle contains a minimal, standalone reproduction path for interactive SAE feature steering on `Qwen/Qwen3.5-35B-A3B-Base`.

Included:

- Interactive multi-layer SAE clamp REPL.
- Zero-default launcher with all registered targets set to `0`.
- Asset downloader for the base model and five SAE layers.
- Vast/cloud bootstrap helper.
- Feature map docs for the currently registered layer/feature pairs.
- Reproducibility and troubleshooting notes.

Not included:

- Model weights.
- SAE weights.
- Hugging Face tokens.
- Vast.ai credentials or SSH configuration.
- A license file. Choose one before public publication.
